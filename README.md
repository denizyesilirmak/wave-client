# Wave-Client

A internal tool for collaboration and communication between team members.
Main goal of wave is to provide a simple and easy to use tool for team members to communicate and collaborate on projects.

## Features

- [ ] Chat
- [ ] Voice Chat
- [ ] Video Chat
- [ ] Screen Sharing
- [x] Themes
- [ ] Custom Emojis
- [ ] Code Snippets

## Installation

```bash
git clone 
cd wave-client
npm install
npm run dev
```
